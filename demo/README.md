# 房地产ETF资产选择平台

## 项目概述

这是一个基于React与Web3技术的房地产ETF资产选择平台，是计算机科学与技术专业的本科毕业设计项目。该平台旨在为用户提供便捷的房地产投资服务，通过区块链技术实现资产代币化和去中心化交易。

## 功能特色

### 🏢 资产展示
- 全球优质房地产资产展示
- 多维度筛选和搜索功能
- 详细的资产信息和收益分析
- 实时价格更新和投资建议

### 📊 投资组合管理
- 个性化投资组合构建
- 智能资产配置建议
- 实时收益追踪和分析
- 风险评估和优化建议

### 💼 交易中心
- 去中心化资产交易
- 智能合约自动执行
- 低手续费和即时结算
- 安全透明的交易记录

### 📈 数据分析
- 专业的市场分析工具
- 收益率计算和预测
- 风险指标监控
- 投资报告生成

## 技术架构

### 前端技术栈
- **HTML5/CSS3**: 现代化的响应式布局
- **JavaScript ES6+**: 交互逻辑和数据处理
- **现代CSS**: 变量、Grid、Flexbox、动画效果
- **响应式设计**: 支持各种设备屏幕

### Web3集成
- **Ethers.js**: 区块链交互库
- **MetaMask**: 钱包连接和管理
- **Smart Contracts**: 智能合约集成
- **IPFS**: 去中心化数据存储

### 区块链技术
- **Ethereum**: 主要区块链网络
- **ERC-20**: 代币化标准
- **DeFi协议**: 去中心化金融集成

## 快速开始

### 环境要求
- 现代浏览器 (Chrome, Firefox, Safari, Edge)
- MetaMask钱包插件
- 稳定的网络连接

### 本地运行
1. 克隆项目到本地
2. 使用任意Web服务器打开项目
3. 访问 `index.html` 开始使用

### 使用说明
1. **连接钱包**: 点击右上角"连接钱包"按钮
2. **浏览资产**: 在资产展示页面查看可投资的房地产ETF
3. **构建组合**: 选择合适的资产构建个人投资组合
4. **执行交易**: 在交易中心完成买入/卖出操作
5. **监控收益**: 在投资组合页面追踪投资表现

async function connectWallet() {

    }
}
